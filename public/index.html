Assignment

Web App Maker

Allows users to create and use online, mobile friendly, Websites. We will be building a Website to create Websites. The website is based on MEAN stack using the four underpinning technologies MongoDB, Express, AngularJS, and Node.js. Users that visit the Websites created by the developers will be referred to as end users. Only developers can create and modify the Websites. End users can not modify the Websites, they can only interact with them

Box Shop: Online Shopping Website


Developed an e-commerce shopping website on MySQL for customers to search, review and purchase products online using JSP and JPA technologies.

The online system uses Walmart Open API for product search using keywords, viewing product details, product reviews and ratings and PayPal Express Checkout API to accept payments.

Additionally registered customers can also maintain a shopping cart to place products of interest, review products, place orders and view their order history
